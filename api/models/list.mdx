---
title: List models
description: Retrieve all models in your tenant
---

List all model instances configured in your tenant. Returns models with their connected endpoints.

## Response

Returns an array of model objects.

<ResponseField name="id" type="uuid">
  Unique identifier for the model.
</ResponseField>

<ResponseField name="name" type="string">
  Model name.
</ResponseField>

<ResponseField name="dtype" type="string">
  Model type identifier (e.g., `openai`, `anthropic`, `ollama`).
</ResponseField>

<ResponseField name="configuration" type="object">
  Model-specific configuration settings.
</ResponseField>

<ResponseField name="summary" type="string">
  Brief description of the model.
</ResponseField>

<ResponseField name="tags" type="string">
  Comma-separated tags.
</ResponseField>

<ResponseField name="created_at" type="datetime">
  Timestamp when the model was created.
</ResponseField>

<ResponseField name="connected_endpoints" type="array">
  Array of endpoints using this model.
  
  <Expandable title="Endpoint properties">
    <ResponseField name="connected_endpoints[].id" type="uuid">
      Endpoint unique identifier.
    </ResponseField>
    
    <ResponseField name="connected_endpoints[].name" type="string">
      Endpoint name.
    </ResponseField>
    
    <ResponseField name="connected_endpoints[].slug" type="string">
      Endpoint URL-friendly slug.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://your-server.com/api/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://your-server.com/api/v1/models",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

models = response.json()
for model in models:
    print(f"{model['name']} ({model['dtype']}) - {len(model['connected_endpoints'])} endpoints")
```
</RequestExample>
