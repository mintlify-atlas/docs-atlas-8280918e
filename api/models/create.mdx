---
title: Create model
description: Create a new model instance with configuration
---

Create a new model instance in Syft Space. Models provide AI/ML capabilities through different providers like OpenAI, Anthropic, Ollama, or vLLM.

## Request

<ParamField path="name" type="string" required>
  Unique name for the model within your tenant.
</ParamField>

<ParamField path="dtype" type="string" required>
  Model type identifier. Use the [list model types](/api/models/types#list-model-types) endpoint to see available types.
  
  Common values:
  - `openai` - OpenAI and OpenAI-compatible APIs
  - `anthropic` - Anthropic Claude models
  - `ollama` - Ollama local models
  - `vllm` - vLLM inference server
</ParamField>

<ParamField path="configuration" type="object" required>
  Model-specific configuration matching the schema for the selected `dtype`. Each model type requires different configuration fields.

  <Expandable title="OpenAI configuration">
    <ParamField path="configuration.api_key" type="string" required>
      OpenAI API key for authentication.
    </ParamField>
    
    <ParamField path="configuration.model" type="string">
      Model identifier to use for completions. Defaults to `gpt-3.5-turbo`.
      
      Examples: `gpt-4`, `gpt-4-turbo`, `gpt-3.5-turbo`
    </ParamField>
    
    <ParamField path="configuration.base_url" type="string">
      Custom base URL for OpenAI-compatible APIs. Defaults to `https://api.openai.com/v1`.
    </ParamField>
    
    <ParamField path="configuration.system_prompt" type="string">
      Default system prompt to prepend to all chat completions.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField path="summary" type="string">
  Brief description of the model's purpose or use case.
</ParamField>

<ParamField path="tags" type="string">
  Comma-separated tags for organizing and filtering models.
  
  Example: `openai,gpt-4,assistant`
</ParamField>

## Response

<ResponseField name="id" type="uuid">
  Unique identifier for the created model.
</ResponseField>

<ResponseField name="name" type="string">
  Model name.
</ResponseField>

<ResponseField name="dtype" type="string">
  Model type identifier.
</ResponseField>

<ResponseField name="configuration" type="object">
  Model configuration as provided in the request.
</ResponseField>

<ResponseField name="summary" type="string">
  Model description.
</ResponseField>

<ResponseField name="tags" type="string">
  Comma-separated tags.
</ResponseField>

<ResponseField name="created_at" type="datetime">
  Timestamp when the model was created.
</ResponseField>

<ResponseField name="updated_at" type="datetime">
  Timestamp when the model was last updated.
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://your-server.com/api/v1/models",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "name": "gpt-4-assistant",
        "dtype": "openai",
        "configuration": {
            "api_key": "sk-...",
            "model": "gpt-4",
            "base_url": "https://api.openai.com/v1",
            "system_prompt": "You are a helpful AI assistant."
        },
        "summary": "GPT-4 model for general assistance",
        "tags": "openai,gpt-4,assistant"
    }
)

model = response.json()
print(f"Created model: {model['id']}")
```
