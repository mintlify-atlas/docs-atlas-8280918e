---
title: List datasets
description: Retrieve all datasets for the current tenant
---

Retrieve a list of all datasets configured for the authenticated tenant, including provisioner status and connected endpoints.

## Request

<ParamField path="GET" type="string">
  `/datasets/`
</ParamField>

### Authentication

Requires authentication via tenant credentials.

## Response

Returns an array of dataset items:

<ResponseField name="id" type="string">
  Unique identifier (UUID) for the dataset
</ResponseField>

<ResponseField name="name" type="string">
  Dataset name
</ResponseField>

<ResponseField name="dtype" type="string">
  Dataset type name (e.g., `local_file`, `remote_weaviate`)
</ResponseField>

<ResponseField name="summary" type="string">
  Brief description of the dataset
</ResponseField>

<ResponseField name="tags" type="string">
  Comma-separated tags
</ResponseField>

<ResponseField name="configuration" type="object">
  Full dataset configuration including type-specific settings
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of creation
</ResponseField>

<ResponseField name="connected_endpoints" type="array">
  List of endpoints connected to this dataset

  <Expandable title="items">
    <ResponseField name="id" type="string">
      Endpoint unique identifier
    </ResponseField>

    <ResponseField name="name" type="string">
      Endpoint name
    </ResponseField>

    <ResponseField name="slug" type="string">
      Endpoint URL slug
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="provisioner_status" type="object" nullable>
  Provisioner status information (for local dataset types)

  <Expandable title="properties">
    <ResponseField name="status" type="string">
      Current provisioner status: `stopped`, `starting`, `running`, `stopping`, or `error`
    </ResponseField>

    <ResponseField name="error" type="string" nullable>
      Error message if status is `error`
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://your-domain.com/datasets/ \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
